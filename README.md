# apg-ai-bias
AI Bias Explorer - APG Project

## Overview
Small proof-of-concept to explore gender bias in Generative AI.

## Features
- Generate images via Hugging Face (FLUX.1-schnell), OpenAI (DALL-E 3), etc.
- Save observations to Supabase.
- Realtime dashboard.

## Setup
App requires Supabase keys in Environment Variables.
